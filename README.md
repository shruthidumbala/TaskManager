# Task Manager

A Node.js + Express + MongoDB task manager project.

## Features
- User authentication
- Task creation, editing, deleting
- Real-time updates with Socket.io
- Role-based access (Admin / User)
- Responsive UI with EJS templates

## Installation

cd TaskManager

##Install dependencies
